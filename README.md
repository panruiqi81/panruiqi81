## 📁 项目展示：图片自动下载器 (Image Downloader)
这是一个基于 Python 的自动化工具，能够快速抓取并下载指定网页的所有图片。

### ✨ 功能特点
- **自动创建文件夹**：下载的图片会自动归档。
- **反爬伪装**：内置 User-Agent 模拟真实浏览器访问。
- **路径自适应**：自动处理相对路径和绝对路径的转换。

### 🛠 使用技术
- Python 3.12
- Requests (网络请求)
- BeautifulSoup4 (网页解析)
